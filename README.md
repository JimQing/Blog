### How To Run Project
```
$ sudo gem install jekyll
```

You'll also need to install the bundler package:

```
$ sudo gem install bundler
```

Download theme and install the dependencies

```
$ bundle install
```

Use it!

```
$ bundle exec jekyll serve -w
```

Use for production!

```
$ bundle exec jekyll build
```
